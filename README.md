# راهنمای تازه‌واردین به واترلو (Waterloo Newcomer Guide)

یک وب‌سایت استاتیک و فارسی‌زبان برای دانشجویان و تازه‌واردین ایرانی به واترلو، انتاریو، کانادا — شامل حمل‌ونقل، گواهینامه رانندگی، محل اقامت، امور مالی، بیمه، مدارک، خدمات دانشگاه واترلو و زندگی روزمره.

محتوای این راهنما گردآوری شده و در این مخزن به یک سایت ساده و قابل جست‌وجو تبدیل شده است.

## ساختار پروژه

```
index.html                  صفحهٔ اصلی سایت (متادیتا، هدر، منوی راست، محل نمایش محتوا)
content/guide.md             متن راهنما به‌صورت مارک‌داون
assets/
  css/style.css               استایل‌ها (راست‌به‌چپ، پالت طلایی/مشکی واترلو، منوی واکنش‌گرا)
  js/markdown.js               مبدل سبک مارک‌داون به HTML (بدون وابستگی خارجی)
  js/app.js                    واکشی guide.md، رندر آن، ساخت منو و اسکرول نرم
  logo.svg, favicon.svg        نشان برگ‌افرا + نشانگر مکان (وکتور)
  favicon-32.png, favicon-180.png, og-image.png   خروجی‌های رستری نشان
```

## اجرای محلی

چون `index.html` محتوای `content/guide.md` را با `fetch` بارگذاری می‌کند، باز کردن مستقیم فایل با پروتکل `file://` کار نمی‌کند (به‌دلیل محدودیت CORS مرورگرها). به‌جایش یک سرور محلی سادهٔ اجرا کنید:

```bash
python -m http.server 8000
```

سپس `http://localhost:8000` را باز کنید. همین سایت را می‌توان بدون هیچ تغییری روی GitHub Pages هم منتشر کرد.
